# Branching
- Main -> for prod release
- Dev -> base branch for development, merges to main
- feat -> new feature / resolved gitHub issue ```dev/feat-describe-your-feature```
- fix -> hotfix ```fix/hotfix-name```
- content -> new blogpost or author or tag added anything content related ```dev/content-whatever-you-add```
